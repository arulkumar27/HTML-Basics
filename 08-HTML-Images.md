# HTML Images

## Overview

Images help make web pages attractive and informative.

## Image Tag

```html
<img src="image.jpg" alt="Sample Image">
```

## Important Attributes

### src

Specifies image location.

```html
<img src="logo.png">
```

### alt

Alternative text if image fails to load.

```html
<img src="logo.png" alt="Company Logo">
```

### Width and Height

```html
<img src="image.jpg" width="300" height="200">
```

## Image from Internet

```html
<img src="https://example.com/image.jpg">
```

## Clickable Image

```html
<a href="index.html">
    <img src="logo.png" alt="Logo">
</a>
```

## Best Practices

* Always provide alt text.
* Optimize image size.
* Use descriptive filenames.
* Avoid large images.

## Real-Time Example

Amazon product pages use images to display:

* Product previews
* Product galleries
* Zoom views

## Interview Questions

### Which tag is used for images?

```html
<img>
```

### Why is alt text important?

It improves accessibility and helps when images fail to load.
