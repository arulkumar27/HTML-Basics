# HTML Audio and Video

## Overview

HTML5 allows developers to embed audio and video directly into web pages without external plugins.

## Audio Element

### Example

```html
<audio controls>
    <source src="song.mp3" type="audio/mpeg">
</audio>
```

## Video Element

### Example

```html
<video width="500" controls>
    <source src="video.mp4" type="video/mp4">
</video>
```

## Useful Attributes

### Controls

```html
controls
```

Displays playback controls.

### Autoplay

```html
autoplay
```

Starts automatically.

### Loop

```html
loop
```

Repeats media continuously.

### Muted

```html
muted
```

Starts without sound.

### Poster

```html
poster="thumbnail.jpg"
```

Displays an image before video playback.

## Real-Time Example

YouTube-like platforms use video elements to stream educational and entertainment content.

Music streaming websites use audio elements to play songs and podcasts.

## Interview Questions

### Which tag is used to embed audio?

```html
<audio>
```

### Which tag is used to embed video?

```html
<video>
```
