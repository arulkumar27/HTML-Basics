# HTML Basics

## 📖 Overview

HTML (HyperText Markup Language) is the standard markup language used to create and structure web pages. Every website on the internet relies on HTML to define its content and layout.

## 🌍 Real-World Usage of HTML

HTML is used in:

* Company Websites
* E-Commerce Platforms
* Blogs and News Portals
* Web Applications
* Landing Pages
* Portfolio Websites
* Documentation Sites

Popular websites such as Google, Amazon, Facebook, and YouTube use HTML as the foundation of their web pages.
