# HTML Elements and Tags

## What are HTML Tags?

Tags are predefined keywords enclosed within angle brackets.

Example:

```html
<h1>Welcome</h1>
```

## HTML Element Structure

```html
<tagname>Content</tagname>
```

Example:

```html
<p>This is a paragraph.</p>
```

## Types of Tags

### Container Tags

Require opening and closing tags.

```html
<h1>Heading</h1>
<p>Paragraph</p>
```

### Empty Tags

Do not require closing tags.

```html
<br>
<hr>
<img src="image.jpg">
```

## Nested Elements

Elements can be placed inside other elements.

```html
<div>
    <h1>Welcome</h1>
    <p>Learning HTML</p>
</div>
```

## Real-Time Example

A navigation bar contains multiple nested links inside a menu container.

## Interview Questions

### What is the difference between a tag and an element?

Tag:

```html
<p>
```

Element:

```html
<p>Hello World</p>
```

An element consists of opening tag, content, and closing tag.
