# HTML Attributes

## What are Attributes?

Attributes provide additional information about HTML elements.

They are written inside opening tags.

## Syntax

```html
<tagname attribute="value">
```

## Common Attributes

### id

```html
<p id="intro">Welcome</p>
```

Unique identifier for an element.

### class

```html
<p class="content">HTML Basics</p>
```

Used to group multiple elements.

### href

```html
<a href="https://example.com">Visit</a>
```

Specifies link destination.

### src

```html
<img src="logo.png">
```

Specifies image location.

### alt

```html
<img src="logo.png" alt="Company Logo">
```

Alternative text for accessibility.

### title

```html
<p title="Information">Hover Here</p>
```

Displays tooltip text.

## Real-Time Example

An online shopping website uses:

* src for product images
* href for product links
* class for styling products

## Interview Questions

### Why are attributes used?

Attributes provide additional information and functionality to HTML elements.

### Which attribute is used for images?

The src attribute specifies the image source.
