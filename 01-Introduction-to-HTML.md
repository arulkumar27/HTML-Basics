# Introduction to HTML

## What is HTML?

HTML (HyperText Markup Language) is the standard markup language used to create and structure web pages. It defines the content and layout of a webpage using various elements and tags.

HTML is not a programming language. Instead, it is a markup language that tells browsers how content should be displayed.

## Why HTML is Important

* Creates webpage structure
* Displays text, images, videos, and links
* Forms the foundation of every website
* Works with CSS and JavaScript
* Essential for web development

## How HTML Works

1. Developer writes HTML code.
2. Browser reads the HTML file.
3. Browser converts HTML into a visual webpage.
4. Users interact with the webpage.

## Real-Time Example

When you open an e-commerce website, HTML creates:

* Product titles
* Product images
* Navigation menus
* Search bars
* Buttons

Without HTML, websites would have no visible content.

## Example

```html
<!DOCTYPE html>
<html>
<head>
    <title>My First Page</title>
</head>
<body>
    <h1>Welcome to HTML</h1>
    <p>This is my first webpage.</p>
</body>
</html>
```
