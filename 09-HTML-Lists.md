# HTML Lists

## Overview

Lists help organize information in a structured manner.

## Unordered List

Displays bullet points.

```html
<ul>
    <li>HTML</li>
    <li>CSS</li>
    <li>JavaScript</li>
</ul>
```

## Ordered List

Displays numbered items.

```html
<ol>
    <li>Install VS Code</li>
    <li>Create HTML File</li>
    <li>Run in Browser</li>
</ol>
```

## Description List

```html
<dl>
    <dt>HTML</dt>
    <dd>Markup Language</dd>

    <dt>CSS</dt>
    <dd>Styling Language</dd>
</dl>
```

## Nested List

```html
<ul>
    <li>Frontend
        <ul>
            <li>HTML</li>
            <li>CSS</li>
        </ul>
    </li>
</ul>
```

## Real-Time Example

E-learning platforms use lists to display:

* Course modules
* Lessons
* Topics

## Interview Questions

### Which tag creates numbered lists?

```html
<ol>
```

### Which tag creates bullet lists?

```html
<ul>
```
