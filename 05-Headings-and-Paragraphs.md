# Headings and Paragraphs

## HTML Headings

HTML provides six heading levels.

```html
<h1>Main Heading</h1>
<h2>Sub Heading</h2>
<h3>Section Heading</h3>
<h4>Topic Heading</h4>
<h5>Small Heading</h5>
<h6>Tiny Heading</h6>
```

## Importance of Headings

* Organize content
* Improve readability
* Improve SEO
* Help screen readers

## Best Practice

Use only one H1 tag per webpage.

## HTML Paragraphs

Paragraphs are created using the p tag.

```html
<p>
This is a paragraph.
</p>
```

## Line Break

```html
<br>
```

Moves text to the next line.

Example:

```html
Hello<br>
World
```

## Horizontal Rule

```html
<hr>
```

Creates a horizontal separator.

Example:

```html
<h1>Section One</h1>
<hr>
<h1>Section Two</h1>
```

## Real-Time Example

A news website uses:

* H1 for article title
* H2 for section headings
* Paragraphs for article content
* HR tags to separate sections

## Interview Questions

### Which heading tag has the highest priority?

```html
<h1>
```

### Why should only one H1 tag be used?

It helps search engines understand the main topic of the webpage and improves SEO.
