# HTML Semantic Elements

## Overview

Semantic elements clearly describe their meaning to browsers, developers, and search engines.

## Benefits

* Better SEO
* Better Accessibility
* Easier Maintenance
* Improved Readability

## Common Semantic Elements

### Header

```html
<header>
    Website Header
</header>
```

### Navigation

```html
<nav>
    Menu Links
</nav>
```

### Main

```html
<main>
    Main Content
</main>
```

### Section

```html
<section>
    About Us
</section>
```

### Article

```html
<article>
    Blog Post
</article>
```

### Aside

```html
<aside>
    Related Content
</aside>
```

### Footer

```html
<footer>
    Copyright Information
</footer>
```

## Complete Example

```html
<header>
    <h1>My Website</h1>
</header>

<nav>
    Home | About | Contact
</nav>

<main>
    <section>
        <h2>About Us</h2>
        <p>Company Information</p>
    </section>
</main>

<footer>
    © 2026 My Website
</footer>
```

## Real-Time Example

News websites use semantic elements to organize:

* Articles
* Categories
* Navigation Menus
* Sidebars

## Interview Questions

### What is Semantic HTML?

Semantic HTML uses meaningful tags that describe content structure.

### Why use Semantic HTML?

It improves SEO, accessibility, and code readability.
