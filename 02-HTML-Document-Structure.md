# HTML Document Structure

## Overview

Every HTML page follows a standard structure that helps browsers understand and render content correctly.

## Basic Structure

```html
<!DOCTYPE html>
<html>
<head>
    <title>My Website</title>
</head>
<body>
    <h1>Hello World</h1>
</body>
</html>
```

## Components

### DOCTYPE Declaration

```html
<!DOCTYPE html>
```

Defines the document type and HTML version.

### HTML Tag

```html
<html>
</html>
```

The root element of the webpage.

### Head Section

```html
<head>
</head>
```

Contains metadata such as:

* Title
* CSS files
* JavaScript files
* SEO information

### Body Section

```html
<body>
</body>
```

Contains all visible content.

## Real-Time Example

For an Amazon product page:

* Head contains title and metadata.
* Body contains product details, images, and buttons.

## Interview Questions

### What is the purpose of the head tag?

The head tag stores metadata and information not directly visible on the webpage.

### What is the purpose of the body tag?

The body tag contains all content visible to users.
