# HTML Links

## Overview

Links allow users to navigate between web pages, websites, and resources.

## Anchor Tag

```html
<a href="https://example.com">Visit Website</a>
```

## External Link

```html
<a href="https://google.com">Google</a>
```

## Open Link in New Tab

```html
<a href="https://google.com" target="_blank">
    Open Google
</a>
```

## Internal Link

```html
<a href="about.html">About Us</a>
```

## Email Link

```html
<a href="mailto:admin@example.com">
    Send Email
</a>
```

## Telephone Link

```html
<a href="tel:+911234567890">
    Call Us
</a>
```

## Download File Link

```html
<a href="resume.pdf" download>
    Download Resume
</a>
```

## Real-Time Example

A company website navigation menu contains:

* Home
* About
* Services
* Contact

All connected using anchor tags.
