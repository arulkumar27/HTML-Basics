# Chapter 06: Text Formatting Tags

In HTML, we use formatting tags to make text look different and to provide semantic meaning.

## Common Tags
- **Bold**: <strong>Important</strong> or <b>Bold</b>
- *Italic*: <em>Emphasized</em> or <i>Italic</i>
- Highlight: <mark>Marked</mark>
- Subscript: H<sub>2</sub>O
- Superscript: X<sup>2</sup>
