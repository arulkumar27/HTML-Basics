# Text Formatting Tags

## Overview

HTML provides various text formatting tags that help improve readability and emphasize important content.

## Bold Text

```html
<b>This text is bold</b>
```

## Strong Text

```html
<strong>This text is important</strong>
```

## Italic Text

```html
<i>This text is italic</i>
```

## Emphasized Text

```html
<em>This text is emphasized</em>
```

## Underlined Text

```html
<u>This text is underlined</u>
```

## Highlighted Text

```html
<mark>Highlighted Text</mark>
```

## Deleted Text

```html
<del>Old Price ₹1000</del>
```

## Subscript Text

```html
H<sub>2</sub>O
```

## Superscript Text

```html
10<sup>2</sup>
```

## Real-Time Example

E-commerce websites use:

* Strong tags for important offers
* Del tags for old prices
* Mark tags for promotions

## Interview Questions

### What is the difference between `<b>` and `<strong>`?

`<b>` makes text bold visually.

`<strong>` makes text bold and indicates importance to search engines and screen readers.
